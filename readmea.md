# Task 02 – Descriptive Statistics (Pure Python, Pandas, Polars)

## Overview
This project performs descriptive statistical analysis on a large political social media ads dataset using three different approaches:

- Pure Python (standard library only)
- Pandas
- Polars

The goal is to understand how different data processing tools handle the same dataset in terms of usability, performance, and complexity.

---

## Project Structure

- `pure_python_stats.py` → Pure Python implementation (no external libraries)
- `pandas_stats.py` → Pandas-based analysis
- `polars_stats.py` → Polars-based analysis
- `requirements.txt` → Required dependencies
- `REFLECTION.md` → Comparison and learning insights
